# Uni GPA Predictor

To use the app, bookmark and use this link: https://usb-tonynot.github.io/UniGPAPredictor/

Σύμβουλος Σπουδών & Στρατηγικής
Οργανώστε την ακαδημαϊκή σας πορεία με το απόλυτο εργαλείο πρόβλεψης βαθμολογίας.

Εισάγετε τους βαθμούς σας για να δείτε άμεσα τον τρέχοντα Μέσο Όρο σας, καθώς και τις Προοπτικές Καριέρας που αντιστοιχούν στο επίπεδό σας.

💡 Στρατηγική Πτυχίου: Μπορείτε να αλλάξετε τις "Προοπτικές Καριέρας" στο μενού για να δείτε ακριβώς τι μέσο όρο χρειάζεστε στα εναπομείναντα μαθήματα ώστε να φτάσετε στο επίπεδο που στοχεύετε.

Πληροφορίες & Συμβατότητα:

💾 Save & Import: Αποθηκεύστε την πρόοδό σας σε αρχείο JSON και φορτώστε την ανά πάσα στιγμή για συνέχεια.
💻 Desktop Only: Η εφαρμογή λειτουργεί βέλτιστα σε Windows/Mac χρησιμοποιώντας Chrome ή Edge για την πλήρη υποστήριξη του συστήματος αρχείων (File System API).
